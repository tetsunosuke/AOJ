標準入力を扱う方法
=========================

コマンドラインの場合は
---------------------------------------
```
$ echo 2 | php <script>
```

ファイルを使う場合
---------------------------------------

```
$ cat data | php <script>
```

or

```
$ php <script> < data
```

PHPスクリプト側ではSTDINを使う
================================

一行なら
---------

```
<?php
$line = fgets(STDIN);
```

複数行なら
------------

```
<?php
while($line=fgets(STDIN))
{
    // 必要な処理
}
```
